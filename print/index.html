GitHub Pull Requests

Pull Request are a way through which developers collaborate on GitHub.

They provide a user-friendly web interface for discussing proposed changes before integrating them into the official project.

Pull requests lets you tell others about changes you've pushed to a GitHub repository.

Once a pull request is sent, interested parties can review the set of changes, discuss potential modifications, and even push follow-up commits if necessary.
